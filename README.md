### Setup & Installtion

Make sure you have the latest version of npm installed.

```bash
git clone <repo-url>
```

```bash
npm install
```

### Running The App
For both frontend & backend
```bash
npm start
```

### Viewing The App
For frontend Go to `http://localhost:3000/` & For backend Go to `http://localhost:8080/`
